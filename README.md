## Install

Maven

```xml
<dependency>
    <groupId>io.github.abudanov</groupId>
    <artifactId>spring-ai-yandex-spring-boot-starter</artifactId>
    <version>0.1.2</version>
</dependency>
```

Gradle
```groovy
implementation 'io.github.abudanov:spring-ai-yandex-spring-boot-starter:0.1.2'
```
